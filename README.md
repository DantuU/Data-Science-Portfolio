### 1. Análisis Exploratorio de Datos (EDA) - "House Prices" ✅
* **Objetivo:** Entender las variables que afectan el precio de una vivienda, limpiar datos nulos con lógica de negocio y detectar valores atípicos (outliers).
* **Herramientas:** Python, Pandas, Matplotlib, Seaborn.
* **Archivo:**[house_prices.ipynb]
* **Logro clave:** Identificación de las 3 variables con mayor correlación al precio (OverallQual, GrLivArea, GarageCars) y limpieza estructurada del dataset.
